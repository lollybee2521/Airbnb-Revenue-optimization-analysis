# Airbnb-Revenue-optimization-analysis

## Project Overview

This project analyzes Airbnb data to help a client identify the most profitable locations and periods to rent out Airbnb properties. The analysis was conducted in Tableau using multiple datasets to uncover revenue trends, pricing patterns, and occupancy opportunities that can support data-driven investment decisions.

## Business Problem

The client aims to maximize revenue from Airbnb rentals and requires insights into where and when properties should be listed for optimal performance. This project leverages Airbnb data to identify key factors influencing revenue generation and rental demand.

## Dataset Description

The analysis was conducted using three separate datasets:

* **Listings Worksheet** – Contains information about Airbnb properties, including listing details, location, room type, and pricing.
**Calendar Worksheet** – Contains daily availability and pricing information for each listing.
**Reviews Worksheet** – Contains customer review information and feedback related to Airbnb listings.

## Data Preparation and Modeling

A critical part of this project involved preparing and combining data from multiple sources within Tableau.

To create a comprehensive dataset for analysis:

The **Listings** worksheet and **Calendar** worksheet were combined using an **Inner Join** in Tableau.
The join was performed using:

**Listing ID** from the Listings worksheet
**ID** from the Calendar worksheet
* These fields served as the primary keys for establishing the relationship between the two datasets.

The use of an Inner Join ensured that only records with matching listing identifiers in both datasets were included in the analysis, resulting in a clean and reliable dataset for visualization and reporting.

This data modeling process enabled the integration of property information with daily availability and pricing data, allowing for deeper analysis of occupancy trends and revenue opportunities.
